Personal Website (This site)

This personal project was my endeavour into HTML, CSS, and Javascript. Prior to this sites creation, I had no knowledge of the aforementioned styling and programming languages. Apart from creating this portfolio page to inform employers about my qualification, I was also able to brush up on linear algebra and differential equations with the creation of the animation above. The animation is based on a multivariate differential equation known as the Halvorsen Chaotic Attractor. I used SVG (Scalable Vector Graphics) to create an approximation of the curves drawn by the Halvorsen equation. Since the equation describes a three-dimensional curve, I employed techniques from linear algebra to rotate the curve and project it onto a two-dimensional screen.
